It’s fantastic to hear that you’re developing new courses tailored to meet the diverse needs of your students at Johns Hopkins University. The breakdown into beginner, intermediate, and advanced courses is a sensible way to cater to different skill levels. Here are some suggestions that may assist you in refining your course proposals:

General Suggestions#

Pre-Assessment:#

Before the course begins, offer a self-assessment or quiz for students to evaluate their own skill levels. This can help them identify which course would be most appropriate for them.

Peer Collaboration:#

Allow students to work in peer-programming sessions. This is an excellent way for beginners to learn from those who are slightly more advanced, and for advanced students to solidify their understanding by teaching concepts.

Course Project:#

In each course, include a culminating project that combines all the skills learned. This will help in the practical application of skills.

Course 1: Introduction to Stata Programming#

Topics to Cover:#

  • Introduction to Stata interface

  • Basic Data Manipulation: Importing, Exporting, Cleaning Data

  • Data Visualization Basics

  • Basic Descriptive Statistics

  • Introduction to Statistical Tests

  • Writing Simple Scripts

Skill Development:#

  • Ability to conduct basic statistical analyses

  • Ability to prepare data for analysis

Deliverables:#

  • Weekly assignments

  • Final project focused on data management and basic analysis

Course 2: Intermediate Stata Programming#

Topics to Cover:#

  • Conditional Statements and Loops

  • Advanced Data Manipulation: Reshape, Merge, Collapse

  • Advanced Statistical Methods: Regression Models, ANOVA

  • Debugging Techniques

  • Macro and Local Programming

Skill Development:#

  • Ability to build and evaluate statistical models

  • Mastery over advanced data manipulation techniques

Deliverables:#

  • Bi-weekly assignments

  • Final project focused on model building and evaluation

Course 3: Advanced Stata Programming#

Topics to Cover:#

  • Automation and Batch Processing

  • Stata’s Mata programming language for Matrix operations

  • Advanced Data Visualization Techniques

  • Simulation Techniques

  • Creating Custom Functions and Programs

  • Version Control and Collaborative Coding in Stata

Skill Development:#

  • Ability to automate repetitive tasks

  • Ability to simulate data for statistical method assessment

Deliverables:#

  • Bi-weekly assignments

  • Final project focused on creating a complex data simulation or automated analysis pipeline

Outreach and Promotion#

Once you’ve structured your courses, consider collaborating with other departments to expand the courses’ reach. For example, you might work with the School of Medicine or the Sociology Department, where many students might need data analysis skills but lack programming knowledge.

Feel free to adapt these ideas as you see fit. Good luck with your course development!
